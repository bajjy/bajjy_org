I'm **Bajjy** — engineer and operator behind a small studio building web platforms,
game-tech, and experimental interfaces.

I've been shipping on the web since the days when "framework" meant "less than 4kb".
A lot of my work since then has been about doing more with less: ES6 templates without
a build step, browser game engines without canvas, automation tools that fit in one file.

The studio takes selective product work — usually greenfield, usually with somebody who
has a clear idea and needs an engineer who can run with it. Get in touch via the
**OPEN_CHANNEL** sector.
